Informations

Les matchs sont récupérés et mis à jour de manière automatique via le site de Riot Games LOLESPORTS.

Un bot discord fait les rappels des matchs la veille de chaque journée, pour les compétitions (paramétrable pour les gestionnaires de groupe).
Pour paramétrer le bot sur un serveur discord, contacter @ggdae.

NOUVEAU : Il peut également vous MP sur discord ! Rendez-vous dans votre profil, en haut à droite.

Les pronostics pour un match sont verrouillés soit :

  • à l'heure annoncée (que le match ait commencé ou non)
  • lorsque le match commence ("en cours" sur lolesports)
  • s'il existe un match précédent : lorsque le match précédent a eu son résultat.
